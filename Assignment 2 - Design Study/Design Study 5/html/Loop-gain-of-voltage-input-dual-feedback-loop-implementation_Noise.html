Noise

Since perfect feedback is nonenergic, the noise could be as low as a simple CS stage (only voltage input is shown, but the same applies for current input). Then the same design equations apply:

$\frac{4 KF_{N18} \eta_{0}}{C_{ant}} < S_{pink max}$

$\frac{5.333 N_{s N18} T k \left(IC_{XU1} + 0.75\right)}{C_{ant} f_{T XU1} \pi \left(IC_{XU1} + 1\right)} < S_{white max}$

Given that the simple CS can satisfiy these equations, this topology must as well.
