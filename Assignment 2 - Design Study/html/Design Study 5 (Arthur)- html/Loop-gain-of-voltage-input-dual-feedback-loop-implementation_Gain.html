Request for feedback

The nice thing about this analysis is that it takes into account the transfer from the source to the driving quantity and from the amplifier output to the load. However, it doesn't facilitate adressing performance limitations for the controller, since there is no straightforward correspondence between the parameters of the transistor and the resulting performance. I thought of finding the chain matrix of the entire system in terms of the chain matrix of the controller, which could be done by subtracting the feedback matrix from it. However, this doesn't give nice expressions and finding the transmission 1 parameters corresponding to a given performance isn't straightforward either. Is there a more insightful way to go about this?
