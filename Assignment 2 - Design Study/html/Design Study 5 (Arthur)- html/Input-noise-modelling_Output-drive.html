Output drive

Because of the dual-loop topology, the CS stage can present the correct output impedance whilst directly driving the load. With ideal feedback networks, the feedback networks require no power, and the output drive behaviour is the same as for a simple CS stage (voltage input is shown, but the same applies for current input). Hence, the same design equations apply:

$I_{b} > \sqrt{\frac{P_{o max}}{R_{load}}}$

$V_{b} > \sqrt{P_{o max} R_{load}}$
