Input noise modelling

Table of contents

    The input stage will dominate the noise performance of the amplifier, since the input-reffered noise contributions of the other stages are divided by its gain. The goal of this page is to find showstopper values for its characteristics based on the input-reffered noise. To this end, we consider a simple input-reffered noise model:

    Towards design equations

    In section, we use the symbolic equations of the noise model to establish necessary conditions for the input transistor. To this end, we'll start by some manipulation of the input-reffered noise PSD:

    Which can be restated as:

    \begin{equation} S_{av}=\frac{0.3333 N_{s N18} T k \left(C_{ant} + c_{iss XU1}\right)^{2} \left(4 IC_{XU1} + 3\right) \left(\alpha f_{T XU1} + f\right)}{\pi C_{ant}^{2} c_{iss XU1} f f_{T XU1} \left(IC_{XU1} + 1\right)} \end{equation}

    In which:

    \begin{equation} \alpha=\frac{3 \pi KF_{N18} \eta \left(IC_{XU1} + 1\right)}{N_{s N18} T_{XU1} k_{XU1} \left(4 IC_{XU1} + 3\right)} \end{equation}

    With:

    \begin{equation} \eta=\frac{c_{iss XU1}}{C_{OX N18} L W} \end{equation}

    $\eta$ is not fully dependent on the dimensions. It can be expressed as:

    \begin{equation} \eta=\frac{75.59 \pi CGBO_{N18}}{W} + \frac{75.59 \pi CGSO_{N18}}{L} + 37.79 \pi C_{OX N18} - \frac{12.6 \pi C_{OX N18}}{N_{s N18}} > \frac{37.79 \pi C_{OX N18} N_{s N18} - 12.6 \pi C_{OX N18}}{N_{s N18}} \end{equation}

    Using this $\eta=\eta_0$, an lower bound for the noise is obtained:

    \begin{equation} S_{av}=\frac{0.3333 N_{s N18} T k \left(C_{ant} + c_{iss XU1}\right)^{2} \left(4 IC_{XU1} + 3\right) \left(\alpha f_{T XU1} + f\right)}{\pi C_{ant}^{2} c_{iss XU1} f f_{T XU1} \left(IC_{XU1} + 1\right)} \end{equation}

    There is an optimum value for $c_{issXU1}=C_{ant}$. Then the minimum noise is:

    \begin{equation} S_{av}=\frac{1.333 N_{s N18} T k \left(4 IC_{XU1} + 3\right) \left(\alpha f_{T XU1} + f\right)}{\pi C_{ant} f f_{T XU1} \left(IC_{XU1} + 1\right)} \end{equation}

    Noise requirements

    The required noise envelope can be decomposed in two components: a frequency-independent white thermal noise component and a frequency-dependent red noise component. The noise model also has a frequency-indepedent and a frequency-dependent component. If either of these components is higher than the associated noise envelope component, the circuit won't meet the specifications. Hence, we can separate these out.

    From the specifications, the thermal noise should not exceed $8.1e-19\,\mathrm{W}\mathrm{Hz}^{-1}$ and the multiplier of the $1/f$ noise should not exceed $3.2400000000000003e-13\,\mathrm{W}\mathrm{Hz}^{-1}$.

    Isolating the components

    $S_{av thermal} > \frac{5.333 N_{s N18} T k \left(IC_{XU1} + 0.75\right)}{\pi C_{ant} f_{T XU1} \left(IC_{XU1} + 1\right)}$

    $S_{av pink} > \frac{5.333 N_{s N18} T \alpha k \left(IC_{XU1} + 0.75\right)}{\pi C_{ant} f \left(IC_{XU1} + 1\right)}$

    Showstopper equations

    $$S_{av pink} f>\frac{4.0 KF_{N18} T \eta k}{C_{ant} T_{XU1} f k_{XU1}}=\frac{2.49539436207407 \cdot 10^{-33}}{T_{XU1} k_{XU1}}<3.24 \cdot 10^{-13}$$

    \begin{equation} S_{av thermal} > \frac{5.333 N_{s N18} T k \left(IC_{XU1} + 0.75\right)}{\pi C_{ant} f_{T XU1} \left(IC_{XU1} + 1\right)}=\frac{4.746 \cdot 10^{-9}}{f_{T XU1}} < 8.1 \cdot 10^{-19} \end{equation}

    From the thermal noise PSD, we find:

    $$ f_{TXU1} > 5.85965431014692\,\mathrm{GHz}$$

    However, the minimum $1/f$ is already higher than the maximum $1/f$ noise, which is a showstopper. Hence, this application cannot be fulfilled in this process without transformers or frequency shifts.
