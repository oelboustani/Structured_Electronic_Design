CS Drive Stage

Introduction

The active antenna is loaded with 50 $\Omega$ coaxial cable.Given a 1 dB compression level at the output of the amplifier of 0 dBm, means an output power of 1 mW.Therefore, the output rms voltage is $V_o= \sqrt{P \cdot R_o} = \sqrt{1\;mW \cdot 50\;\Omega} = 0.2236 V$The output rms current is $I_o = V_o/R_o = 0.2236 V / 50\;\Omega = 4.472 mA$From these values we get the peak voltage $V_p = \sqrt{2} \cdot 0.2236 = 316.2 mV$ and the peak value of the current is $I_p = 6.324 mA$. The biasing voltage $V_{DS}$ should be therefore equal to the peak voltage and the biasing current $I_D$ should be equal to the peak current. Furthermore, the antenna is modeled as voltage source with a capacitor in series.
